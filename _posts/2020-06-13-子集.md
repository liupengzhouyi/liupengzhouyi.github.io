# [子集](https://leetcode-cn.com/problems/subsets/)

## Title

![截屏2020-06-14 上午12.01.00](https://tva1.sinaimg.cn/large/007S8ZIlly1gfr3uoqc9ej31040p2ac8.jpg)

## MyCode

```c++
		vector<vector<int>> subsets(vector<int>& nums) {
        vector<vector<int>> res;
        for(int item : nums){
            int len=res.size();
            //首先保存自身
            res.push_back({item});
            //之前的子集加上自身后保存
            for(int i=0; i<len; ++i){
                auto temp=res[i];
                temp.push_back(item);
                res.push_back(temp);
            }
        }
        res.push_back({});
        return res;
    }
```
